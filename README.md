## RocketMan
A Hangman Game For People Who Like Space!!

## Requirements
This game is an iOS app that requires iOS 10.2+ to run. 

## Installation
Download this project and run using the latest version of Xcode.

## Technical Highlights
* Xcode 9.3 
* Written in Swift 4.1
